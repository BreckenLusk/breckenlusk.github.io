Here are some of my latest projects:

Saily

I codeveloped, translated, and marketed Saily, a modern, fast, and beautiful package manger for jailbroken iOS and iPadOS. You can follow the official Saily account on Twitter for occasional updates.

InternalStore

This is project of mine in which I document Apple's internal applications, firmware, images, and occasionally other leaks. You can visit my page on The iPhone Wiki to view the full extent of my research.

InternalEnabler

This is a script for jailbroken iOS devices which enables certain features that are present in InternalUI builds without having to install the full InternalUI build, improving stability and performance.

macCSS

This is a website which emulates modern macOS in a web browser using CSS, HTML5, and JavaScript. Originally developed for a school project, I will be updating it soon to match the latest version of macOS.
